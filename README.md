# 💍 WeddingPro

A comprehensive wedding management platform that enables couples to create beautiful, interactive wedding pages where they can share their love story, collect RSVPs, and curate photo memories from their special day.

## ✨ Features

- 🔐 **Secure Authentication** - Email-based OTP authentication system
- 💕 **Beautiful Wedding Pages** - Romantic, animated public wedding pages with floating hearts
- 📸 **Photo Management** - Unlimited guest photo uploads with approval workflow
- 📝 **RSVP System** - Complete RSVP management with approval workflow
- 💖 **Love Story Timeline** - Custom relationship timeline creation
- 🎨 **Modern Design** - Mobile-responsive with elegant animations and gradients

## 🚀 Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS v4 with custom animations
- **Backend**: Supabase (PostgreSQL + Auth + Storage + Real-time)  
- **Deployment**: Vercel/Netlify ready

## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd weddings-pro
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Add your Supabase credentials
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```

5. **Visit** `http://localhost:5173`
